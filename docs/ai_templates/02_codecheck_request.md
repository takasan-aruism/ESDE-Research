## 使い方

- 中量 / 重量作業で必須。軽量は省略
- 中量はチェック項目セクション (A 〜 F) のうち**改変範囲のみ**抽出
- 重量は全セクション
- 各チェック項目は **チェックボックス形式** で出す。Code B はそれに ✓ / ✗ で答える

## 不承認時の対応

- 出力不完整 → 該当 seed を単独再実行依頼
- 集計不整合 → 再集計スクリプト提供を依頼、diff 確認
- 不変量逸脱 → Code A に巻き戻し依頼、原因調査
- 解釈の不一致 → Taka に戻して指示仰ぐ
