# Code B 承認/不承認レポートテンプレ

*用途*: Code B が依頼書 (02_codecheck_request) に答える応答書
*ファイル名規則*:
- 承認: `<phase_id>_<work_name>_check_approved.md`
- 不承認: `<phase_id>_<work_name>_check_denied.md`
*配置*: 依頼書と同じディレクトリ

---

## 使い方

- 依頼書のチェック項目 (A〜F) をそのまま見出しに使う
- 各項目に「✓ / ✗ / 該当せず」と**根拠** (実コマンド出力 / 計算結果 / grep 結果) を併記
- 結論 (承認 / 不承認) は冒頭で明示
- **何もチェックしていない承認は出さない**。各項目の検証ログを残す
