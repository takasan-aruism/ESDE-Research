## 8. 本格 run 構成

| 項目 | short | long |
|---|---|---|
| seeds | 48 (0..47) | 5 (0..4) |
| maturation | 20 | 20 |
| tracking | 10 | 50 |
| window_steps | 500 | 500 |
| 並列 | parallel -j24 | parallel -j5 |

```bash
# short
seq 0 47 | parallel -j24 python <script>.py \
  --seed {} --tracking-windows 10 --tag short

# long
seq 0 4 | parallel -j5 python <script>.py \
  --seed {} --tracking-windows 50 --tag long
```

`OMP/MKL/OPENBLAS_NUM_THREADS=1` 必須。
