# Phase 完了報告書テンプレ

*用途*: 相談役 Claude が Taka に提出する Phase 結果レポート
*ファイル名規則*: `<phase_id>_<work_name>_report.md` (例: `v912_l06_timeseries_report.md`) または `<phase_id>_phase_summary.md`
*配置*: 対象作業と同じディレクトリ (例: `primitive/v9XX/`)

---

## 使い方

- **数値は表で出す**。本文に値を散らさない
- **「観察」と「解釈」を分節する**。観察は数値、解釈は推論
- 解釈には**反例 / 別解釈の可能性**を併記
- §キー発見サマリは**観察由来のみ**。解釈の主張を入れない
- 「次フェーズへの示唆」では複数案を並べる、相談役判断で勝手に絞らない
