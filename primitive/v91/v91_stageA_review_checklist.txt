SECTION B: gamma 負の安全性
================================================================

[ ] gamma < 0 の場合の M 計算:
    gamma=-0.5, ratio=2.0 → M = 1.0 + (-0.5)×(2.0-1.0) = 0.5 → clamp → 0.8
    gamma=-0.5, ratio=0.5 → M = 1.0 + (-0.5)×(0.5-1.0) = 1.25 → clamp → 1.2
    gamma=-1.0, ratio=2.0 → M = 1.0 + (-1.0)×(2.0-1.0) = 0.0 → clamp → 0.8
    gamma=-1.0, ratio=0.5 → M = 1.0 + (-1.0)×(0.5-1.0) = 1.5 → clamp → 1.2

    REVIEWER: clamp が安全弁として機能。
    gamma=-1.0 でも M は [0.8, 1.2] に収まる。
    系が即座に崩壊することはない。

    GPT §4.3: 負gamma は物理層へ波及しうる。
    torque弱化 → θ同期低下 → リンク維持低下 → alive_links低下。
    間接的だが物理層に影響する。観測対象。

[ ] gamma=0.0 の場合:
    M = 1.0 + 0.0×(ratio-1.0) = 1.0（常に）。
    v8.5 相当（フィードバックなし）。baseline。
